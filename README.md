# clj-hector

A simple Clojure client for Cassandra that wraps Hector

## Usage

    (def c (cluster "Pauls Cluster" "localhost"))
    (def ks (keyspace c "Twitter"))
    (get-rows ks "Users" ["paul"])

    (-> (cluster "Pauls Cluster" "localhost")
        (keyspace "Twitter")
        (get-rows "Users" ["paul"]))
